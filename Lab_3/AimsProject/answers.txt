6. What are the advantages of Polymorphism?

Answer:

Allows code reusability
Improves flexibility and scalability
Enables dynamic method binding (runtime behavior)
Makes code easier to maintain and extend
7. How is Inheritance useful to achieve Polymorphism in Java?

Answer:
Inheritance allows subclasses to inherit methods from a superclass and override them. This enables polymorphism, where a superclass reference can point to subclass objects and call overridden methods dynamically.

8. What are the differences between Polymorphism and Inheritance in Java?

Answer:

Inheritance: mechanism for code reuse (parent-child relationship)
Polymorphism: ability to use one interface for different data types
Inheritance is a structural concept, while polymorphism is behavioral
Polymorphism often relies on inheritance to work
